•steps to create a api key in anthropic 
 1. go to anthropic api console on :- console.anthropic.com
 2. click on get the api key button in the api console dashboard
 3. click on create api key 
 4. select or create the workspace(workspace is basically a segmentaion provide by anthropic  for our api keys with the help of workspace we can categories the api keys and we can also define different attributes for workspace which will help us to control the access of the api key on basis of these such attributes are 
 spending limit: - we can set monthly spend cap for all keys in workspace collectivly by setting the cap for workspace, we can manage model access permission with workspace for api key by defining which model can be accessed by workspace, Team Member access:- define kar sakte hai konsa user/ member can access the workspace,and workspace can also be used to track the token consumptions)
 5. add the name for the api key
 6.click on add 
 7. copy the key and use it
